Webparts360

Our web IDE for SharePoint Online

Webparts360 is our in-house framework: a web IDE for SharePoint Online that lets us build business applications from three configurable building blocks — input, process, and output — so we can deliver multi-state, document-heavy workflows faster while keeping performance and upgrades predictable.

Configure, don’t constantly recode

Use XML settings, HTML for UI, and T-SQL for transaction logic to configure forms, processes, and views instead of rebuilding each app from scratch.

Built for every SharePoint version
