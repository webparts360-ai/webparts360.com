Course Content

Structured, hands-on agenda designed for busy professionals

Morning

Foundations, Data Prep & Modeling

  • Introduction: Power BI vs Excel, Power BI Desktop, Service & Mobile
  • Navigating the Power BI Desktop interface and key settings
  • Importing data from Excel and other common sources
  • Cleaning and transforming data with Power Query
  • Handling missing data, duplicates, and simple data quality issues
  • Building a basic data model with lookup tables and relationships
Afternoon

DAX, Time Intelligence & Dashboards

  • Understanding calculated columns vs measures in DAX
  • Creating core business measures: revenue, profit, margin
  • Adding time-intelligence: last month, year-to-date, growth %
  • Designing interactive report pages with charts, tables, and slicers
  • Publishing to Power BI Service and creating a simple dashboard
  • Setting up scheduled refresh using a personal gateway
